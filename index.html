Capítulo 1: Polinomio de Taylor

(1.1) Polinomio de Taylor (1D) f \left( x \right) \simeq f \left( x_0 \right) + f' \left( x_0 \right) \left( x - x_0 \right) + \frac{1}{2!} f'' \left( x_0 \right) \left( x - x_0 \right)^2 + \frac{1}{3!} f''' \left( x_0 \right) \left( x - x_0 \right)^3 + ... x \simeq x_0

Capítulo 2: Diagonalización

(2.1) Diagonalizar una matriz A es
encontrar una base de vectores: \left\{ \vec{v}_1, \vec{v}_2 \right\} que cumplan: A \vec{v}_1 = \lambda_1 \vec{v}_1 A \vec{v}_2 = \lambda_2 \vec{v}_2 siendo: \lambda_1, \lambda_2 \equiv \text{ valores propios } \vec{v}_1, \vec{v}_2 \equiv \text{ vectores propios }
(2.2) Teorema A \text{ simétrica } \Longrightarrow \text{ diagonalizable }
(2.3) Matriz ortogonal es aquella que cumple: M = \left( \fbox{ \begin{matrix} \ \\ \vec{v}_1 \\ \ \end{matrix} } \ \fbox{ \begin{matrix} \ \\ \vec{v}_2 \\ \ \end{matrix} } \right) \text{det } M = \pm 1 \vec{v}_1 \bot \vec{v}_2 \text{ ortogonales} lo cual implica que: M^{-1} = M^{T} \text{Si det } M = + 1 \Longrightarrow M \text{ rotación}
(2.4) Teorema: A \text{ simétrica } \Longleftrightarrow \text{ ortogonalmente diagonalizable } Además: A \text{ simétrica} \Longleftrightarrow \begin{matrix} D = M^T A M \\ A = M D M^T \end{matrix} \begin{pmatrix} \lambda_1 & 0 \\ 0 & \lambda_2 \end{pmatrix} = \begin{pmatrix} \fbox{ \ \ \ \vec{v}_1 \ \ \ } \\ \ \\ \fbox{ \ \ \ \vec{v}_2 \ \ \ } \end{pmatrix} \begin{pmatrix} \\ \ \ \ A \ \ \ \\ \ \end{pmatrix} \left( \fbox{ \begin{matrix} \ \\ \vec{v}_1 \\ \ \end{matrix} } \ \fbox{ \begin{matrix} \ \\ \vec{v}_2 \\ \ \end{matrix} } \right)
